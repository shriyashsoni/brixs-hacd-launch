# Project Profile: Brixs Chain
**Ticker:** BRIXS
**Type:** FT
**Description:** A Unified Liquidity L2/L3 tailored specifically for Emerging Markets (India, Africa, LatAm). It solves the L2 fragmentation problem.

## Utility
- **At Launch:** Pay transaction fees on the testnet L2 (gasless via account abstraction paymasters).
- **Planned:** Mainnet deployment, L3 expansion, MEV protection rewards.
