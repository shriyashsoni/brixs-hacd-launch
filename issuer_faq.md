# Issuer FAQ: Brixs Chain
**Q: What is the formation cost reference?**
A: Each lot requires stacking 1 HACD. With a 50 HAC reference cost per HACD, the cost is around 50 HAC + network fees.

**Q: What is BRIXS used for?**
A: It is intended for paying transaction fees on the Brixs L2 testnet, with future plans for mainnet utility.
